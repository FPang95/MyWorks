List of type combinations by abundance

From Bulbapedia, the community-driven Pokémon encyclopedia.
Jump to: navigation, search

The abundance of Pokémon type combinations runs the gamut from the very common Normal type to unique combinations such as Ice/Ghost or pure Flying.

There are 324 possible ways to assign types to Pokémon (18 single types and 18 * 17 dual types). However, the game makes no distinction between a Pokémon's first and second types when calculating damage; that is, a Pokémon which is Dark/Ghost is treated the same as one that is Ghost/Dark. Therefore, instead of considering every permutation, this list will only consider type combinations, of which there are 171 (18 single and 18 * 17 / 2 dual). This means that Pokémon such as Spiritomb and Sableye will be treated as having the same type.

For the purposes of this list, each form with a different type than another of the same Pokémon's forms counts as a different Pokémon. For example, Darmanitan counts once toward pure Fire and also once toward Fire/Psychic, while Deoxys only counts once toward Psychic.

Of the total 171 possible type combinations, 25 have not yet been assigned to any Pokémon. 29 type combinations have only been assigned to a single Pokémon, and a further 28 have been assigned to multiple Pokémon within the same evolutionary family. The other 88 type combinations have been assigned to multiple unrelated Pokémon.

In the table below, each column counts the number of Pokémon of that type introduced in that generation (with some exceptions). The Total column counts all Pokémon in the National Pokédex as of Generation VII.

In the Generation I column below, Magnemite and Magneton are considered Electric/Steel even though they did not gain the Steel type until Generation II. In the Generation IV column, Rotom's forms are each counted separately even though they were not assigned separate types until Generation V. All of the pre-Generation VI Pokémon that were given the Fairy type are also counted as Fairy types rather than as their original types for their generation. For convenience, Arceus and Silvally count only toward the Normal type.
